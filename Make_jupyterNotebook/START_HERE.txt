================================================================================
 LangGraph Notebook Annotator - Educational Knowledge Graph Builder
================================================================================

Welcome! This project automatically annotates Jupyter notebooks with markdown 
comments and builds interactive knowledge graphs for educational use.

QUICK START (2 minutes)
=======================

1. Install dependencies:
   pip install -r requirements.txt

2. Verify env.txt has OPENAI_API_KEY=sk-proj-...

3. Run the agent:
   python agent.py --notebook practice_notebook.ipynb --output-dir output

4. View results:
   - Annotated notebook: output/annotated_notebook.ipynb
   - Interactive graph: output/graph_interactive.html
   - PNG visualization: output/knowledge_graph.png

FILE STRUCTURE
===============

agent.py                  Main entry point (run this!)
nodes/                    Pipeline components
  read_notebook.py        → Parse .ipynb
  generate_comments.py    → LLM comment generation
  annotate_notebook.py    → Inject markdown
  build_graph.py          → Construct knowledge graph
  export_graph.py         → Create visualizations

utils/                    Extended features
  semantic_search.py      → Natural language graph queries
  ast_analyzer.py         → Variable dependency tracking
  quiz_generator.py       → Auto-generate quiz questions
  interactive_viz.py      → HTML/Cytoscape visualization

output/                   Generated files (created at runtime)
  annotated_notebook.ipynb
  knowledge_graph.json
  knowledge_graph.png
  graph_interactive.html
  ... and more

EXAMPLE USAGE
==============

# Basic
python agent.py --notebook my_notebook.ipynb

# With custom output directory
python agent.py --notebook analysis.ipynb --output-dir docs/

# Run feature showcase
python demo.py

KEY FEATURES
=============

✓ Automatic markdown comment generation using OpenAI LLM
✓ Knowledge graph with hierarchical relationships (IS_BELONG_TO, IS_NEXT_TO)
✓ Multiple export formats: JSON, PNG, HTML, NetworkX, Cytoscape
✓ Semantic search over graph using natural language
✓ Variable dependency analysis using Python AST
✓ Auto-generated quiz questions for assessment
✓ Interactive HTML visualization with physics layout
